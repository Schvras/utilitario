Teclas de Atalho

Utilize as teclas Ctrl + Alt do seu teclado junto a algum número, confome os itens abaixo:

Ctrl + Alt + B - Foco no campo de pesquisa

Ctrl + Alt + H - Home

Ctrl + Alt + D - Developer

Ctrl + Alt + E - Evolution

Ctrl + Alt + F - Flow

Ctrl + Alt + M - Manager

Ctrl + Alt + P - Plus

Ctrl + Alt + S - Smart

Ctrl + Alt + L - Lista Ramais (Somente na Home)
